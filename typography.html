Type Scale

To create and design an easily readable interface, you don't need lots of font-sizes. A simple ratio-based scale with 7-8 options should suffice.

Font scale is based on the default browser size of 16px and an 8px grid.

Using an even number like 8 to size and space elements makes scaling for a wide variety of devices easy and consistent.

  • 48px (3rem)
  • 36px/2.25rem
  • 24px/1.5rem
  • 20px/1.25rem
  • 16px/1rem
  • 14px/.875rem
  • 12px/.75rem

Line Height

  • Copy - 1.5 (150%)
  • Title - 1.2 (120%)
